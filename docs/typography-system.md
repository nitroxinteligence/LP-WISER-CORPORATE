- **Font Family**: Inter, system-ui, -apple-system, sans-serif

## Acessibilidade

### Contraste Mínimo
- Texto principal (#ffffff) sobre fundo (#040609): **21:1** ✅
- Texto secundário (rgba(255,255,255,0.8)) sobre fundo: **16.8:1** ✅
- Texto terciário (rgba(255,255,255,0.6)) sobre fundo: **12.6:1** ✅

### Leiturabilidade
- **Line Height**: Mínimo 1.5 para corpo de texto
- **Line Length**: Máximo 75 caracteres por linha
- **Paragraph Spacing**: 1.5x-2x font size

## Implementação CSS/Tailwind

### Variáveis CSS Customizadas
```css
:root {
  /* Font Families */
  --font-headings: 'Manrope', system-ui, -apple-system, sans-serif;
  --font-body: 'Inter', system-ui, -apple-system, sans-serif;

  /* Font Weights */
  --font-regular: 400;
  --font-medium: 500;
  --font-semibold: 600;
  --font-bold: 700;
  --font-extra-bold: 800;

  /* Letter Spacing */
  --tracking-tight: -0.02em;
  --tracking-normal: 0;
  --tracking-wide: 0.01em;
  --tracking-wider: 0.08em;
}
```
